Css

First off

  • To add style in the html file, use the <style> attribute
  • For a seperate css file, link the .css file in the head.

Color:

  1. #RRGGBB with the hexadecimal numbers.
    • Red: #FF0000
    • Green: #00FF00
    • Blue: #0000FF
    • White: #FFFFFF
  2. rgb(R, G, B), each ranges from 0-225.
    • Red: rgb(225, 0, 0)
    • Green: rgb(0, 255, 0)
    • Blue: rgb(0, 0, 255)F
    • Black: rgb(0, 0, 0)
  3. rgba where a is for transparency, ranging from 0.0-1.0.
    4. (colorpicker.com)

Background:

    (inside the body tag)
  • for color: #F3D8C0
  • for picture: url(link)
    • background-repeat: no-repeat; for just a single frame of the picture.
    background-size: cover; to fill the body.

Border:

  • width
  • color
  • style
    • solid
    • dotted
    • double
    • dashed
  • Example: border: 5px solid purple

Selectors:

  • id (can be used only one)
  • class
  • Star (*)
  • Descendant Selector (change selectors inside other selectors)
  • Example: todoList.
